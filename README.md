# Blood Bank Management System

A web application for managing blood donors, blood inventory, and hospital requests using **Java Servlets & MVC architecture**.

 Features
- Manage Blood Inventory (Add, Update, Delete, View)
- Donor Registration & Donation History
- Blood Request Management
- JSON File-Based Data Storage
- MVC Pattern with Service & DAO Layers

 Tech Stack
- Java 11+, JSP, JSTL, Servlets
- Maven
- Apache Tomcat 9+
- JSON Storage

 Run the Project

```bash
mvn clean package
mvn tomcat7:run
